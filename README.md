# 民生警务

> 本项目是民生警务和高德地图接入的二合一项目

## 编译安装

``` bash
# 安装全部依赖
npm install

# 启动本地开发环境 localhost:8080
npm run dev

# 编译线上环境
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
